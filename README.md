# Vidflix

Small Netflix clone built with React/Redux during Brian Holt's workshop.

## Features

- React
- Redux
- Webpack
- Yarn
- React-Router
- Testing with Jest/Enzyme
- ES6
- Universal rendering
- Preact

## Instructions

- `git clone`
- `npm run dev`
- open `localhost:8080`
